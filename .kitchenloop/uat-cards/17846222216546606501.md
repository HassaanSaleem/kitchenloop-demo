# User Test Card: 17846222216546606501 (batch: create-validation / search / share)

## Feature (one sentence)
A user of the relay-notes HTTP API can create notes (with bad input rejected
by an honest 400), find notes by a case-insensitive search across title, body,
and tags, and share only notes that actually exist.

## Prerequisites
- [ ] Node.js is installed (this project has zero runtime dependencies — no install step)
- [ ] Commands are run from the repository root (the folder containing `src/server.mjs`)

## Prerequisite Verification Commands
```bash
node --version            # should print a v20+ (or newer) version string
ls src/server.mjs         # should print: src/server.mjs
```

## Steps

### Step 1: Create a well-formed note
```bash
node --input-type=module -e '
import { buildApp } from "./src/server.mjs";
import { mkdtempSync } from "node:fs"; import { tmpdir } from "node:os"; import { join } from "node:path";
const app = buildApp({ storePath: join(mkdtempSync(join(tmpdir(), "uat-")), "notes.json") });
await new Promise(r => app.listen(0, r));
const base = `http://127.0.0.1:${app.address().port}`;
const res = await fetch(base + "/notes", { method: "POST", body: JSON.stringify({ title: "Relay Launch", body: "kickoff plan", tags: ["work"] }) });
const note = await res.json();
app.close();
if (res.status === 201 && note.id) console.log("STEP 1 PASS status=" + res.status + " id=" + note.id);
else { console.log("STEP 1 FAIL status=" + res.status + " body=" + JSON.stringify(note)); process.exit(1); }
'
```

**Expected output contains:** "STEP 1 PASS status=201"

### Step 5: Share an existing note and read it back through the token
```bash
node --input-type=module -e '
import { buildApp } from "./src/server.mjs";
import { mkdtempSync } from "node:fs"; import { tmpdir } from "node:os"; import { join } from "node:path";
const app = buildApp({ storePath: join(mkdtempSync(join(tmpdir(), "uat-")), "notes.json") });
await new Promise(r => app.listen(0, r));
const base = `http://127.0.0.1:${app.address().port}`;
const note = await (await fetch(base + "/notes", { method: "POST", body: JSON.stringify({ title: "Shareable" }) })).json();
const shareRes = await fetch(base + "/notes/" + note.id + "/share", { method: "POST" });
const { token } = await shareRes.json();
const readRes = await fetch(base + "/shared/" + token);
const shared = await readRes.json();
app.close();
if (shareRes.status === 201 && token && readRes.status === 200 && shared.id === note.id) console.log("STEP 5 PASS shareStatus=201 readStatus=200");
else { console.log("STEP 5 FAIL shareStatus=" + shareRes.status + " readStatus=" + readRes.status); process.exit(1); }
'
```
**Expected exit code:** 0
**Expected output contains:** "STEP 5 PASS shareStatus=201 readStatus=200"
**Expected output does NOT contain:** "FAIL"
